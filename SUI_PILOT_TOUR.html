    Reference — the six context-injection mechanisms (deep dive)

    This lab is a guided simulation. If you want the deep dive on each mechanism, the verbatim source snippets, and the file/line citations, expand the panels below. Each section maps to one of the six layers shown in the inspector.

     1 · Always-on @-import

    The user's global ~/.claude/CLAUDE.md contains @~/.claude/sui-pilot/agents/sui-pilot-agent.md. Claude Code resolves that import at session start and pastes the 71-line agent body into the system prompt — for every session, in every directory. This is the only sui-pilot mechanism that loads without user action. Backing file: agents/sui-pilot-agent.md (lines 27–46 hold the Doc-First rule; lines 33–41 hold the topic→corpus routing table).

     2 · Slash command /sui-pilot

    Plugin registers six slash commands under commands/; five are skills (next section), and the sixth, /sui-pilot, is a routing command. The body of commands/sui-pilot.md is appended to the conversation when invoked — a short prologue that hands the work to the specialized subagent. Unlike the @-import, the command body lives in conversation context, not the system prompt.

     3 · The sui-pilot-agent subagent

    The same file from §1 also doubles as a dispatchable subagent. The frontmatter (model: opus, 10 MCP tools, standard editor/shell tools) selects which tools and model a fresh subagent context gets when the Agent tool dispatches with subagent_type: "sui-pilot:sui-pilot-agent". As an @-import the file body lives in the main session; as a subagent it is the system prompt of a fresh context with no memory of the main conversation. /move-pr-review uses this heavily — 10 parallel reviewer subagents + 1 consolidator.

     5 · move-lsp MCP server

    Declared in .claude-plugin/plugin.json (lines 15–21). 10 tools backed by move-analyzer inject runtime context computed from the user's current code: diagnostics, hover types, completions, goto/type definition, references, document symbols, code actions, inlay hints, and rename. The only injection mechanism whose payload is impossible to predict at design time.
